# 经典电视剧投票系统

一个美观、功能完整的经典电视剧投票系统，让用户可以为喜爱的经典电视剧投票，见证时代的记忆。

## 功能特点

### 核心功能
- 📺 **电视剧展示**：以卡片形式展示经典电视剧，包含剧名、年代、主演、海报和票数
- 🔍 **搜索功能**：输入电视剧名称快速定位到相应卡片
- 📅 **年代筛选**：按80年代、90年代、2000年代筛选电视剧
- 🗳️ **投票功能**：为喜爱的电视剧投票，实时更新票数
- 🏆 **排行榜**：按票数排序显示前10名电视剧
- 📊 **票数统计**：显示总票数、最高票数、电视剧总数和前三名
- 💬 **评论功能**：点击电视剧卡片查看和添加评论
- ➕ **添加电视剧**：用户可以添加系统中没有的经典电视剧
- 🎵 **背景音乐**：打开页面时自动播放背景音乐，支持暂停/播放

### 技术特点
- 🎨 **美观界面**：怀旧复古风格，暖色调为主，响应式设计
- 💾 **数据持久化**：使用localStorage存储投票和评论数据
- 🔄 **实时更新**：投票后实时更新票数和排行榜
- 📱 **响应式设计**：适配移动端和桌面端
- ✨ **动画效果**：添加了丰富的动画和过渡效果
- 🎯 **用户友好**：直观的操作界面，流畅的用户体验

## 如何使用

### 本地运行
1. 克隆或下载本项目到本地
2. 进入项目目录
3. 启动本地服务器：
   - 使用Python：`python -m http.server 8000`
   - 或使用其他本地服务器工具
4. 在浏览器中访问：`http://localhost:8000`

### 主要操作
- **投票**：点击电视剧卡片上的"投票"按钮
- **搜索**：在搜索框中输入电视剧名称，点击"搜索"按钮
- **筛选**：点击年代筛选按钮查看对应年代的电视剧
- **添加电视剧**：点击"+ 添加电视剧"按钮，填写表单提交
- **查看评论**：点击电视剧卡片打开评论模态框
- **添加评论**：在评论模态框中输入评论内容，点击"发表评论"按钮
- **控制音乐**：点击页面右上角的音乐控制按钮
